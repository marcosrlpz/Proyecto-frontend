# Portfolio React — Marcos | Frontend

Portfolio profesional desarrollado con **React + Vite**, enfocado a mostrar
proyectos reales consumidos desde la **GitHub API**, con una arquitectura clara,
buenas prácticas y enfoque a contratación **frontend / full stack**.

--------------------------------------------------

## 🚀 Demo

👉 https://proyecto-frontend-c9ze8m3tm-marcos-projects-483d877c.vercel.app/

Deploy en producción con **Vercel**.

--------------------------------------------------

## ✨ Características principales

- Diseño **100% responsive**
- Navegación con **React Router DOM**
- Página de **Proyectos** con datos reales desde la **GitHub API**
- Página de **Detalle de Proyecto** con rutas dinámicas
- **Custom Hook** `useFetch` para manejo de peticiones (loading / error / data)
- **useContext** para gestionar proyectos destacados
- Persistencia de favoritos en **localStorage**
- **Formulario de contacto** con `react-hook-form` y validación
- Arquitectura modular y escalable
- Estilos globales con **styled-components**

--------------------------------------------------

## 🧱 Arquitectura del proyecto

src/  
├─ components/      # Componentes reutilizables (Navbar, Layout, ProjectCard…)  
├─ context/         # Estado global (FavoritesContext)  
├─ hooks/           # Custom hooks (useFetch)  
├─ pages/           # Home, Projects, ProjectDetail, Contact  
├─ services/        # Lógica de servicios / API  
└─ styles/          # Theme y estilos globales  

--------------------------------------------------

## 🧠 Requisitos del proyecto

- Web full responsive

- Mínimo de 3 páginas usando react-router-dom
